# BRENFUSION
is a game-ish-thing-kind-of where you create a fucked up frankenstein of a bunch of brencord member's characters

### it is built on the following open source libraries:
- [p5.js](https://github.com/processing/p5.js) (by [Processing Foundation](https://processingfoundation.org))
- [p5.clickable.js](https://github.com/Lartu/p5.clickable) (by [Lartu](https://lartu.net))
- [SoundJS](https://github.com/CreateJS/SoundJS) (by [CreateJS](https://createjs.com))
- [PreloadJS](https://github.com/CreateJS/PreloadJS) (by [CreateJS](https://createjs.com))
